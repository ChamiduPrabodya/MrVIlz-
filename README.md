# MrVilz Web

Starter workspace with:

- React frontend in `client`
- Node.js + Express backend in `server`
- Root scripts to run both together

## Run locally

```bash
npm install
npm run dev
```

If PowerShell blocks `npm` scripts on your machine, use:

```bash
npm.cmd install
npm.cmd run dev
```

The frontend dev command now opens `http://localhost:5173/` in Microsoft Edge automatically on this Windows setup.

Frontend: `http://localhost:5173`

Backend API: `http://localhost:5000/api/health`

## Production build

```bash
npm run build
npm start
```
